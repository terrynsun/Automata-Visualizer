A deterministic finite automata (DFA) or finite state machine is a machine that takes in letters from a set alphabet and changes states accordingly. By definition, it is: - deterministic: it can only hold one state at a time - finite: it cannot be defined to have infinite states. Finite automata are useful when you only have a very limited amount of memory to work with. It saves its state, but does not need to keep track of everything that has already happened.
